# API: Данные для карточек курсов

Эндпоинт: `GET /api/courses`

Ответ должен быть объектом (или массивом), где каждый курс содержит следующие поля.

---

## Поля одного курса

| Поле        | Тип      | Обязательное | Описание                                                  |
|-------------|----------|:------------:|-----------------------------------------------------------|
| `id`        | integer  | ✅           | Уникальный идентификатор курса                            |
| `slug`      | string   | ✅           | URL-идентификатор курса, используется в роуте `/courses/:slug` |
| `title`     | string   | ✅           | Название курса                                            |
| `excerpt`   | string   | ✅           | Краткое описание (отображается на карточке)               |
| `image`     | string   | ✅           | URL обложки курса (фоновое изображение карточки)          |
| `level`     | string   | ✅           | Уровень сложности: `beginner`, `intermediate`, `advanced` |
| `duration`  | string   | ✅           | Продолжительность курса, например `"6 недель"`            |
| `price`     | string   | ✅           | Цена, например `"4 900 ₽"` или `"Бесплатно"`             |

---

## Ожидаемая структура ответа

Текущий фронтенд ожидает **объект**, где ключи — это `slug` курса:

```json
{
  "blender-cad": {
    "id": 1,
    "slug": "blender-cad",
    "title": "Blender CAD для начинающих",
    "excerpt": "Научитесь 3D-моделированию в Blender с нуля.",
    "image": "https://cdn.example.com/courses/blender.jpg",
    "level": "beginner",
    "duration": "6 недель",
    "price": "Бесплатно"
  },
  "algorithm-rush": {
    "id": 2,
    "slug": "algorithm-rush",
    "title": "Алгоритмы и структуры данных",
    "excerpt": "Освойте алгоритмы и структуры данных для прохождения технических интервью.",
    "image": "https://cdn.example.com/courses/algorithms.jpg",
    "level": "advanced",
    "duration": "8 недель",
    "price": "4 900 ₽"
  }
}
```

> Если бэкенд возвращает массив `[]`, потребуется небольшая адаптация на фронте в `AllCourses.js` и `Home.js`.

---

## Значения поля `level`

Фронтенд использует `level` для CSS-класса бейджа и фильтрации. Допустимые значения:

| Значение       | Отображение на UI |
|----------------|-------------------|
| `beginner`     | Начальный         |
| `intermediate` | Средний           |
| `advanced`     | Продвинутый       |

---

## Примечания

- Поле `description` (полное описание) нужно только на странице курса (`GET /api/courses/:slug`), не на карточке.
- Поля `features`, `program`, `reviews`, `instructors` также нужны только на странице курса, не в списке.
- `image` должен быть абсолютным URL или путём, доступным браузеру — карточка использует его как `background-image`.
