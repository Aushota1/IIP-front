# Дизайн страницы Tasks (по мотивам All Cups)

## Цветовая схема

### Основные цвета
- **Background**: `#20212B` (темный фон)
- **Primary**: `#A6A6C3` (светло-серый)
- **Secondary**: `#B066F0` (фиолетовый акцент)
- **Accent**: `#7A7A99` (серо-синий)
- **Text Primary**: `#FFFFFF` (белый текст)
- **Link**: `#7A7A99` (серо-синий для ссылок)

### Компоненты
- **Input Background**: `#282934`
- **Button Primary Background**: `#1A1C25`
- **Button Border**: `#6A6A81`
- **Border Radius**: `6px` (кнопки), `8px` (общий)

## Типографика

- **Font Family**: VK Sans Display (или альтернатива: Inter, Roboto)
- **Font Size**: `16px` (базовый)
- **Heading Sizes**: `16px` (h1, h2)

## Структура страницы

### Заголовок
```
Тренажер
```

### Панель управления
- Кнопка "Сортировка"
- Кнопка "Фильтры"
- Счетчики:
  - Все подборки: 11
  - В процессе: 0
  - Пройденные: 0

### Сетка карточек задач

Каждая карточка содержит:
- **Изображение** (превью задачи)
- **Количество задач** (например, "347 задач")
- **Бейджи** (например, "Баллы от «Другого Дела»", "Алгоритмы")
- **Заголовок** (название подборки)
- **Описание** (краткое описание подборки)
- **Прогресс** ("0% пройдено")

## Макет компонентов

### Карточка задачи
```
┌─────────────────────────────────┐
│                                 │
│         [Изображение]           │
│                                 │
├─────────────────────────────────┤
│ 347 задач                       │
│                                 │
│ [Бейдж 1] [Бейдж 2]            │
│                                 │
│ Алгоритмы                       │
│                                 │
│ Задачи охватывают более 30...   │
│                                 │
│ 0% пройдено                     │
└─────────────────────────────────┘
```

### Стили карточки
- **Background**: `#282934` (темнее основного фона)
- **Border Radius**: `8px`
- **Padding**: `16px`
- **Hover Effect**: легкое увеличение яркости или тень
- **Transition**: `0.3s ease`

### Бейджи
- **Background**: `#1A1C25`
- **Border**: `1px solid #6A6A81`
- **Border Radius**: `6px`
- **Padding**: `4px 12px`
- **Font Size**: `14px`
- **Color**: `#FFFFFF`

### Прогресс-бар
- **Background**: `#1A1C25`
- **Fill Color**: `#B066F0` (фиолетовый)
- **Height**: `4px`
- **Border Radius**: `2px`

## Адаптивность

### Desktop (> 1200px)
- 3 карточки в ряд
- Gap: `24px`

### Tablet (768px - 1200px)
- 2 карточки в ряд
- Gap: `20px`

### Mobile (< 768px)
- 1 карточка в ряд
- Gap: `16px`

## Интерактивность

### Hover эффекты
- Карточка: `transform: translateY(-4px)`, `box-shadow: 0 8px 24px rgba(176, 102, 240, 0.2)`
- Кнопки: `background: #252631`, `border-color: #8A8A99`

### Анимации
- Появление карточек: `fade-in` с задержкой
- Переходы: `transition: all 0.3s ease`

## Accessibility

- Все интерактивные элементы доступны с клавиатуры
- Контрастность текста соответствует WCAG AA
- Альтернативный текст для изображений
- ARIA-метки для кнопок и ссылок

## Примечания

Дизайн вдохновлен платформой All Cups (cups.online), адаптирован под задачи по программированию с сохранением темной цветовой схемы и современного минималистичного стиля.
