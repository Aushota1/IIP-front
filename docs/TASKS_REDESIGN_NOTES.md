# Точное копирование дизайна All Cups

## Ключевые отличия от текущей реализации:

### 1. Структура страницы
- Заголовок "Тренажер" (крупный)
- Панель поиска и фильтров
- Вкладки с счетчиками (Все подборки, В процессе, Пройденные)
- Сетка карточек

### 2. Поиск и фильтры
- Поиск с иконкой лупы ВНУТРИ инпута (слева)
- Два дропдауна: "Сортировка" и "Фильтры" со стрелками вниз
- Все в одной строке на темном фоне (#282934)

### 3. Вкладки
- Три вкладки с бейджами-счетчиками
- Активная вкладка подсвечена
- Бейджи с числами в кружках

### 4. Карточки задач
**Структура:**
- Изображение сверху (180px высота)
- Бейдж "X задач" поверх изображения (правый верхний угол)
- Бейдж "Баллы от Другого Дела" с иконкой (если есть)
- Контент снизу:
  - Теги (цветные бейджи)
  - Заголовок
  - Описание (3 строки с обрезкой)
  - Футер: круговой прогресс + "X% пройдено"

### 5. Круговой прогресс
- SVG круг (не линейный бар)
- Зеленый цвет для заполнения
- Процент рядом

### 6. Цвета (точные из оригинала)
- Фон страницы: #20212B
- Фон карточек: #282934
- Фон кнопок: #1A1C25
- Бордеры: #6A6A81
- Акцент: #B066F0
- Текст: #FFFFFF

1. Убрать текущую панель управления с кнопками
2. Добавить правильный поиск с иконкой внутри
3. Добавить вкладки с бейджами
4. Переделать карточки: изображение сверху, бейджи поверх
5. Заменить линейный прогресс на круговой SVG
