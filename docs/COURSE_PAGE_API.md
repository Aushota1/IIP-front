# API: Данные для страницы курса

Эндпоинт: `GET /api/courses/:slug`

---

## Поля верхнего уровня

| Поле          | Тип    | Обязательное | Описание                                              |
|---------------|--------|:------------:|-------------------------------------------------------|
| `id`          | int    | ✅           | Уникальный ID курса (используется при записи на курс) |
| `slug`        | string | ✅           | URL-идентификатор                                     |
| `title`       | string | ✅           | Название курса                                        |
| `excerpt`     | string | ✅           | Краткое описание (под заголовком в шапке)             |
| `description` | string | ✅           | Полное описание (вкладка «О курсе»)                   |
| `image`       | string | ✅           | URL обложки курса (фото в шапке страницы)             |
| `level`       | string | ✅           | `beginner` / `intermediate` / `advanced`              |
| `duration`    | string | ✅           | Продолжительность, например `"6 недель"`              |
| `price`       | string | ✅           | Цена, например `"4 900 ₽"` или `"Бесплатно"`         |
| `features`    | array  | ✅           | Преимущества курса (вкладка «О курсе»)                |
| `program`     | array  | ✅           | Программа уроков (вкладка «Программа»)                |
| `reviews`     | array  | ✅           | Отзывы студентов (вкладка «Отзывы»)                   |
| `instructors` | array  | ✅           | Преподаватели (секция под вкладками)                  |

---

## features[]

| Поле    | Тип    | Описание                        |
|---------|--------|---------------------------------|
| `icon`  | string | Эмодзи или символ               |
| `title` | string | Заголовок преимущества          |
| `text`  | string | Краткое описание преимущества   |

---

## program[]

| Поле         | Тип    | Описание                                      |
|--------------|--------|-----------------------------------------------|
| `content_id` | int    | ID урока (используется для отметки прогресса) |
| `title`      | string | Название урока                                |
| `description`| string | Краткое описание урока                        |
| `duration`   | string | Длительность, например `"20 мин"`             |

---

## reviews[]

| Поле     | Тип    | Описание                                      |
|----------|--------|-----------------------------------------------|
| `name`   | string | Имя автора отзыва                             |
| `role`   | string | Должность / роль автора                       |
| `text`   | string | Текст отзыва                                  |
| `rating` | int    | Оценка от 1 до 5                              |
| `avatar` | string | URL фото автора отзыва                        |

---

## instructors[]

| Поле       | Тип    | Описание                              |
|------------|--------|---------------------------------------|
| `name`     | string | Имя преподавателя                     |
| `position` | string | Должность / специализация             |
| `bio`      | string | Краткая биография                     |
| `photo`    | string | URL фото преподавателя                |
| `social`   | array  | Список ссылок на соцсети (может быть пустым `[]`) |

### instructors[].social[]

| Поле   | Тип    | Описание                        |
|--------|--------|---------------------------------|
| `icon` | string | Эмодзи или иконка платформы     |
| `url`  | string | Полный URL профиля              |

---

## Медиафайлы — сводка

| Поле                    | Где используется              | Формат  |
|-------------------------|-------------------------------|---------|
| `image`                 | Обложка в шапке страницы      | URL     |
| `reviews[].avatar`      | Фото автора отзыва            | URL     |
| `instructors[].photo`   | Фото преподавателя            | URL     |

Все URL должны быть абсолютными и доступными браузеру напрямую.
